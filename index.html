CUPCAKE REMEMBER

The entire game was developed by me with Unity engine and C#, with the help of some acquired graphics and sound assets. It was a big challenge to develop a game with a commercial objective on my own, but it made me evolve a lot as a developer.

The game is in the final stages of development, with release scheduled for October on Steam and and in the future for mobile devices (Android and IOS).

CUPCAKE REMEMBER is a game developed for PC, it is a mixture of puzzles and memory game, where the player needs to memorize a sequence of colors and then find the best way to complete the challenges, without forgetting the initial sequence.

To create the game, I looked for inspiration in the game (Simon - Genius) and Bomberman.

Currently the game is in the testing phase with a demo game with 25 levels available. Soon the game will be released on Steam with a total of 50 levels and some other features.

You can download the demo version of the game from the link below.

-For installation, just extract the contents of the .zip file to a folder of your choice and run the game.

CatBox Alive/Dead

CatBox Alive/Dead is a game developed by Paulo Junior and Junior Neuvald to Unamed Game Jam. It is a 2D game where there are two mirrored boxes and the player must control two cats that move in opposite directions.

I was responsible for developing the game's programming in Unity Engine and C#.

Game developed for a GameJam in two days as the main reference in the mental experiment of Schrödinger's cat. You can download the game for testing at the bottom of the page.

Swool

I was responsible for developing the game's programming in Unity Engine and C#. It's a simple game, created for a GameJam in two days but with very interesting mechanics.
